Presentazion

It's Friulian for presentation

Design Goals

  • Everything centered
  • Contents as big as possible
  • Scale and display well on any resolution
  • Center source code and left-align it
  • Painless PDF export
  • Single, easy to edit HTML file with
    slide contents

Tech Features

  1. Fast
  2. Cross browser, on fairly recent browsers
  3. Only one dependancy: jQuery (included)
  4. PDF export via wkhtmltopdf
  5. Search
